# Code of Conduct

This repository follows the spirit of Contributor Covenant v2.1.

## Our Pledge

We expect contributors to help keep the project welcoming, respectful, and
safe. Harassment, discrimination, or abusive behavior is not acceptable.

## Standards

Examples of positive behavior include:

- Using inclusive, respectful language.
- Giving and receiving constructive feedback.
- Respecting differing viewpoints and experience levels.

Unacceptable behavior includes:

- Harassing, insulting, or intimidating others.
- Publishing private information without permission.
- Deliberately disrupting discussion or review.

## Enforcement

Project maintainers may remove or moderate content, pause review, or restrict
participation when conduct violates this policy.

## Scope

This policy applies to repository issues, pull requests, comments, reviews, and
any project-maintained communication channels.

## Reporting

If you need to report a conduct issue, open a private GitHub security advisory
or contact the repository maintainers directly through the project's normal
maintainer channel.
