# Ontology mapping and knowledge graph

The mapping graph stores explicit crosswalks between ontology standards. It
exports JSON, JSON Schema, Turtle, JSON-LD, summary statistics, and Mermaid
visualization artifacts under `data/ontologies/`.

Inferred mappings are separate work. Treat close matches and source crosswalks as
inspection aids unless a downstream reviewed rules-as-code workflow upgrades them
to executable semantics.
