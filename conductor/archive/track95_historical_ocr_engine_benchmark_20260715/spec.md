# Track 95: Historical NZ OCR Engine Benchmark

## Goal

Run a zero-cost, reproducible empirical comparison of the four Track 87 engine candidates on rights-cleared historical NZ pages.

## Requirements

- Pin engine code, weights, containers, SBOMs, licences, images, and gold annotations.
- Benchmark Docling, PP-StructureV3, Surya, and olmOCR without replacing source OCR.
- Report CER, WER, layout/order, tables, calibration, disagreement, cost, and throughput.
- Use only no-cost GitHub, Hugging Face, Kaggle, or Colab capacity.

## Acceptance

Reproducible benchmark and model cards support an explicit promote or defer decision with a hard zero-cost gate.
