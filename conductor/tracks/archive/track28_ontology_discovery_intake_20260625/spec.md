# Track 28: Ontology Discovery and Intake

**Dependencies**: Track 25  
**Parallelization Node**: Standards Discovery  
**Status**: Complete

## Goal

Identify additional ontologies, vocabularies, schemas, and standards that improve legislative, parliamentary, policy, legal, graph, vector, or rules-as-code interoperability.

## Scope

- Official standards bodies and public vocabulary registries.
- Government legal and parliamentary metadata standards.
- Legal informatics and rules-as-code literature.
- Graph, vector, provenance, and publication standards that support this repo.

## Acceptance Criteria

- [x] Maintain a discovery log with source, version, license, authority, relevance, and risk.
- [x] Triage candidates as implement, map-only, monitor, or reject.
- [x] Add approved candidates to the standards registry created by Track 26.
- [x] Create implementation blockers for candidates that need unavailable source data.
