# Vale Prose Linter Guidelines (nlp-policy-nz)

This document establishes the guidelines for spelling, grammar, and style checks using the Vale prose linter.

---

## 1. Core Rule Sets

To enforce high-quality, professional documentation, the Vale setup utilizes the following styles:
- **Vale**: Default built-in general spelling and readability rules.
- **Microsoft**: Standard technical writing styles (covers tone, abbreviations, and active voice).
- **Prose**: Custom project rules (such as allowing Māori vocabulary exceptions).

---

## 2. Prose Guidelines

- **Tone**: Keep a professional, objective, and clear tone. Avoid slang or overly casual colloquialisms in official specifications.
- **Active Voice**: Prefer active voice (e.g., "The pipeline processes the text") over passive voice (e.g., "The text is processed by the pipeline").
- **Māori Word Exceptions**: Custom dictionary definitions ensure that Te Reo Māori vocabulary (with or without macrons) does not trigger spelling or style warnings.
